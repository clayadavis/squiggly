squiggly
========

Simple in-browser presentation creator


Installation
====

This repo includes [marked](https://github.com/chjj/marked) as a submodule. To
clone, including the submodules, do

    git clone --recursive <git URL>

or, if you're using an older version of git, you may need to

    git clone <git URL>
    cd squiggly
    git submodule init
    git submodule update
